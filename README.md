# 🎲 Casino Dice Game (Console App)

A simple .NET console-based **Casino Dice Game**.

---

## 📌 Game Description

The player starts with a **credit amount** and plays up to **10 rounds** (or until the credit is exhausted).

### In each round:
- The player places a **bet amount**
- The player chooses **Low (1–3)** or **High (4–6)**
- A **6-sided dice** is rolled
- The game evaluates whether the player **wins or loses**
- If the player wins → **2× bet payout**
- If the player loses → **bet is lost**

---

## ⏹️ Game End Conditions

The game ends when **either**:
- **10 rounds** are completed
- The player **runs out of credit**
